EMB

EMB is an image sharing social network that allows users to create and manage thematic personal box, image collections such as events, interests, hobbies and more.

Packages: 

  - bootstrap-3
  - jquery-masonry
  - moment
  - npm

By Álex Arqués and Pere Torres
